# Simon Says Game

I made this project as a simple memory game using plain HTML, CSS, and JavaScript.

The idea is straightforward: the game flashes a sequence of colors, and you have to repeat that exact sequence by clicking the buttons in order. At each new level, one extra color is added, so you need to remember everything from the start of the pattern.

If you click a wrong color, the game ends, the screen gives quick red feedback, and you can restart right away.

## Run it

Just open `index.html` in your browser.

No setup, install, or build step is needed.

## How to play

1. Press any key to start.
2. Watch the button that flashes.
3. Click the same color.
4. On the next level, watch the longer pattern.
5. Repeat the full sequence from the beginning.
6. If you make a mistake, the game ends and you can restart by pressing any key.

## Score

- Your score is the level you reached.
- The game also shows the highest score during your current session.
